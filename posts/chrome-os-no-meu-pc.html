Chrome OS no meu pc

12 de outubro de 2013Sem categoriaGeorge Moura

Atenção faça os testes usando uma máquina virtual ou um pc que não tenha utilidade

Aí galera venho falar para vocês sobre o Chrome OS, o sistema operacional da Google, acabei de instalar ele no meu pen-drive e já estou utilizando, a propósito estou escrevendo este post nele.

Eu estava empolgado para comprar um chromebook, mas aí como sistema é open source resolvi pesquisar e achei uma iso, então resolvi testar o ditocujo. O sistema é bem simples como de costume da Google, ele na verdade é uma tela com o chrome instalado e mais nada, não existe a pasta “Meus documentos”, “Computador”, “Painel de Controle”, “Acessórios” ou qualquer outra pasta ou aplicativo que você possa imaginar, a única pasta que existe é a “Files” e dentro dela a pasta Downloads e Google Drive  é meus amigos, é simplesmente o chrome e os aplicativos que você instalar e visualizadores (imagem, audio e video), mas só conseguir ver imagens, arquivos mp3 são reconhecidos, mas não consegui reproduzir, nem pdf da pra ver, só baixando algum aplicativo ou pelo gmail.

A sistema que estou utilizando na verdade é o Chromium (a versão open source do Chrome OS), no chromium, os plugins do flash e do pdf reader que já vem integrado ao chrome, não vêm habilitados mas a comunidade já achou um maneira de habilitá-los, veja aqui.

O Chromium OSe o Chrome OS, são sistemas diferentes neste link da pra ver a diferença entre eles, basicamente um da mais trabalho que o outro para configurar, mas fazem a mesma coisas.

A instalação do Chrome OS no pc é bem simples veja aqui o passo a passo: http://chromeos.hexxeh.net/.

Você vai precisar de um pen-drive de 4Gb e depois de copiado a ISO é só dar boot no pc via usb, no link acima você também acha as imagens para VMWare e Virtualbox.
