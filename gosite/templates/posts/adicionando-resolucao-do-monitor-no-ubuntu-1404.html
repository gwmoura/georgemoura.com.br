Adicionando resolução do monitor no ubuntu 14.04

15/12/2015 10:46
Estou escrevendo este post para não esquecer como configurar novas resoluções de monitores no meu ubuntu. O processo é bem simples e possui 3 passos apenas:
  1. Descobrir as configurações ideais para a resolução
  2. Adicionar um novo modo com xrandr
  3. Adicionar os comandos no ~/.bashrc

1 - Descobrir as configurações ideais

Para essa etapa use o comando gtf [altura] [largura] [profundidade] , seguindo o exemplo abaixo você obterá algo desse tipo: 
     $ gtf 1920 1080 60 -x

     # 1920x1080 @ 60.00 Hz (GTF) hsync: 67.08 kHz; pclk: 172.80 MHz
     Modeline "1920x1080_60.00"  172.80  1920 2040 2248 2576  1080 1081 1084 1118  -HSync +Vsync

2 - Adicionar um novo modo com xrandr

Agora copie a saída do comando gtf, copie apenas a linha após a partir da palavra "Modeline" 
      $ xrandr --newmode "1920x1080_60.00"  172.80  1920 2040 2248 2576  1080 1081 1084 1118  -HSync +Vsync
      $ xrandr --addmode VGA1 1920x1080_60.00
      $ xrandr -s 1920x1080

3 - Adicionar os comando no ~/.bashrc

Abra o terminal ctrl+alt+t e digite nano ~/.bashrc, vá até o final do arquivo e copie os comandos: 
      # Monitor Config

      xrandr --newmode "1920x1080_60.00"  172.80  1920 2040 2248 2576  1080 1081 1084 1118  -HSync +Vsync
      xrandr --addmode VGA1 1920x1080_60.00
      xrandr -s 1920x1080
Aperte ctrl+x e depois "S" para salvar e fechar.
Com isso toda vez que você reiniciar o computador as configurações serão carregadas.
